# Netflix Clone 🎬

A responsive Netflix landing page clone built using HTML and CSS.

## 🌐 Live Demo

https://bhavyapro-7.github.io/Netflix-Clone/

## ✨ Features

- Responsive Design
- Netflix-inspired UI
- Sign In Page
- Mobile Friendly
- Modern CSS Layouts
- Hover Effects and Animations

## 🛠 Tech Stack

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries

## 📂 Project Structure

```
Netflix-Clone/
│
├── index.html
├── style.css
├── signin.html
├── signin.css
├── Assets/
└── favicon.ico
```
